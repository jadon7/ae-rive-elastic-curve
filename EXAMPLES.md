## Rive 曲线示例

### Elastic - 弹性曲线

#### 示例 1: 按钮点击反馈

**场景**: 用户点击按钮时的缩放动画

**参数推荐**:
```
amplitude: 1.2
period: 0.3
easingType: easeOut
```

**应用步骤**:
1. 创建按钮图层
2. 在 0s 设置缩放为 100%
3. 在 0.5s 设置缩放为 120%
4. 选中两个关键帧
5. 应用 Elastic (easeOut) 曲线

**效果**: 按钮快速放大，然后带有轻微弹跳效果回到目标大小

---

#### 示例 2: 弹窗出现

**场景**: 对话框从屏幕中心弹出

**参数推荐**:
```
amplitude: 1.5
period: 0.5
easingType: easeOut
```

**应用步骤**:
1. 创建对话框图层
2. 在 0s 设置缩放为 0%，不透明度为 0%
3. 在 0.8s 设置缩放为 100%，不透明度为 100%
4. 分别对缩放和不透明度应用曲线

**效果**: 对话框从无到有，带有明显的弹性效果

---

#### 示例 3: 强调动画

**场景**: 突出显示某个元素

**参数推荐**:
```
amplitude: 2.0
period: 0.8
easingType: easeInOut
```

**应用步骤**:
1. 创建需要强调的图层
2. 在 0s 设置位置为起点
3. 在 1.2s 设置位置为终点
4. 应用 Elastic (easeInOut) 曲线

**效果**: 元素移动时带有夸张的弹性效果，非常吸引注意力

## Android 曲线示例

### 基础曲线

#### Linear - 线性

**典型应用**: 进度条、颜色渐变

**参数**: 无

**示例**: 加载进度条
```
场景: 进度条从 0% 到 100%
关键帧: 0s (0%), 2s (100%)
效果: 匀速增长，适合表示持续的加载过程
```

---

#### Accelerate - 加速

**典型应用**: 物体下落、快速退出

**参数推荐**: `factor: 1.5`

**示例**: 卡片滑出屏幕
```
场景: 卡片从屏幕中央滑出到屏幕外
关键帧: 0s (y=540), 0.5s (y=1200)
效果: 卡片越滑越快，模拟被甩出的感觉
```

---

#### Decelerate - 减速

**典型应用**: 物体停靠、滑动停止

**参数推荐**: `factor: 1.5`

**示例**: 列表滚动停止
```
场景: 列表快速滚动后逐渐停止
关键帧: 0s (y=0), 1s (y=500)
效果: 快速开始，逐渐减速，模拟惯性滑动
```

---

#### AccelerateDecelerate - 加速减速

**典型应用**: 页面切换、自然移动

**参数**: 无

**示例**: 页面水平切换
```
场景: 从页面 A 滑动到页面 B
关键帧: 0s (x=0), 0.8s (x=1920)
效果: 平滑的加速和减速，非常自然
```

---

### 物理曲线

#### Anticipate - 预期

**典型应用**: 投掷动作、蓄力效果

**参数推荐**: `tension: 2.0`

**示例**: 投掷物体
```
场景: 物体被投掷出去
关键帧: 0s (x=100), 1s (x=1000)
效果: 先向后退一点（蓄力），然后快速向前
```

**参数对比**:
- `tension: 1.0` - 轻微后退
- `tension: 2.0` - 明显后退（推荐）
- `tension: 4.0` - 夸张后退

---

#### Overshoot - 超调

**典型应用**: 弹性停靠、强调终点

**参数推荐**: `tension: 2.0`

**示例**: 抽屉打开
```
场景: 侧边抽屉从左侧滑入
关键帧: 0s (x=-300), 0.6s (x=0)
效果: 抽屉滑入后稍微超过目标位置，然后回弹
```

**参数对比**:
- `tension: 1.0` - 轻微超调
- `tension: 2.0` - 明显超调（推荐）
- `tension: 4.0` - 强烈超调

---

#### AnticipateOvershoot - 预期超调

**典型应用**: 夸张动画、游戏效果

**参数推荐**: `tension: 2.0`

**示例**: 弹簧门
```
场景: 门被推开
关键帧: 0s (rotation=0°), 1s (rotation=90°)
效果: 先向后转一点，然后快速打开并超过 90°，最后回弹到 90°
```

---

#### Bounce - 弹跳

**典型应用**: 掉落动画、强调落地

**参数**: 无

**示例**: 球掉落
```
场景: 球从高处掉落到地面
关键帧: 0s (y=100), 1.2s (y=900)
效果: 球落地后多次弹跳，逐渐静止
```

**使用技巧**:
- 适合垂直方向的动画
- 需要足够的时间（建议 1s 以上）才能看到完整的弹跳效果
- 可以配合旋转属性增强真实感

---

### Material Design 曲线

#### FastOutSlowIn - 快出慢入

**典型应用**: 大多数 UI 动画（Material Design 推荐）

**参数**: 无（贝塞尔: 0.4, 0.0, 0.2, 1.0）

**示例**: 卡片展开
```
场景: 卡片从小图标展开为全屏
关键帧: 0s (scale=10%, position=icon), 0.5s (scale=100%, position=center)
效果: 快速开始，缓慢结束，符合 Material Design 规范
```

**为什么推荐**:
- Google 官方推荐的默认曲线
- 适用于 80% 的 UI 动画
- 给人以响应迅速的感觉

---

#### FastOutLinearIn - 快出线性入

**典型应用**: 元素退出屏幕

**参数**: 无（贝塞尔: 0.4, 0.0, 1.0, 1.0）

**示例**: 通知消失
```
场景: 通知卡片向右滑出屏幕
关键帧: 0s (x=0), 0.4s (x=2000)
效果: 快速开始，保持速度离开
```

---

#### LinearOutSlowIn - 线性出慢入

**典型应用**: 元素进入屏幕

**参数**: 无（贝塞尔: 0.0, 0.0, 0.2, 1.0）

**示例**: 菜单展开
```
场景: 菜单从顶部下拉
关键帧: 0s (y=-200), 0.4s (y=0)
效果: 匀速开始，缓慢停止
```

## iOS 曲线示例

### UIView.animate 标准曲线

#### Linear - 线性

**典型应用**: 简单动画、颜色渐变

**示例**: 淡入淡出
```
场景: 图片淡入
关键帧: 0s (opacity=0%), 0.5s (opacity=100%)
效果: 匀速淡入
```

---

#### EaseIn - 缓入

**典型应用**: 元素进入

**示例**: 文字从下方滑入
```
场景: 标题从底部滑入
关键帧: 0s (y=1200), 0.6s (y=540)
效果: 慢速开始，加速进入
```

---

#### EaseOut - 缓出

**典型应用**: 元素退出

**示例**: 按钮按下
```
场景: 按钮被按下时缩小
关键帧: 0s (scale=100%), 0.2s (scale=95%)
效果: 快速开始，缓慢停止
```

---

#### EaseInOut - 缓入缓出

**典型应用**: 平滑过渡

**示例**: 页面切换
```
场景: 页面从右向左切换
关键帧: 0s (x=1920), 0.7s (x=0)
效果: 平滑的加速和减速
```

---

### UIView.animate Spring - 弹簧动画

#### Spring Default - 默认弹簧

**典型应用**: 通用弹性动画

**参数**: 无（内部: damping=0.7, velocity=0.0）

**示例**: 模态窗口出现
```
场景: 模态窗口从中心弹出
关键帧: 0s (scale=80%, opacity=0%), 0.6s (scale=100%, opacity=100%)
效果: 带有适度弹性的出现动画
```

---

#### Spring Gentle - 温和弹簧

**典型应用**: 细腻的 UI 动画

**参数**: 无（内部: damping=0.9, velocity=0.0）

**示例**: 开关切换
```
场景: 开关从关到开
关键帧: 0s (x=0), 0.4s (x=50)
效果: 非常柔和的弹性效果，适合小元素
```

---

#### Spring Bouncy - 弹跳弹簧

**典型应用**: 强调动画、游戏效果

**参数**: 无（内部: damping=0.5, velocity=1.0）

**示例**: 点赞动画
```
场景: 点赞图标放大
关键帧: 0s (scale=100%), 0.5s (scale=150%)
效果: 明显的弹跳效果，非常有趣
```

---

#### Spring Custom - 自定义弹簧

**典型应用**: 需要精确控制的弹簧动画

**参数推荐**:
```
快速响应: damping=0.8, velocity=0.5, duration=0.3
柔和弹跳: damping=0.6, velocity=0.0, duration=0.5
强烈弹跳: damping=0.4, velocity=1.5, duration=0.7
```

**示例**: 自定义按钮反馈
```
场景: 按钮点击后的缩放动画
参数: damping=0.6, velocity=0.8, duration=0.4
关键帧: 0s (scale=100%), 0.4s (scale=110%)
效果: 快速响应，带有明显弹跳
```

---

## 实战案例

### 案例 1: 移动应用启动动画

**需求**: 制作一个 App 启动时的 Logo 动画

**方案**:
1. **Logo 缩放**: 使用 `iOS Spring Bouncy`
   - 0s: scale=0%, opacity=0%
   - 1s: scale=100%, opacity=100%

2. **Logo 旋转**: 使用 `Android AccelerateDecelerate`
   - 0s: rotation=0°
   - 1s: rotation=360°

3. **背景渐变**: 使用 `Linear`
   - 0s: color=#FFFFFF
   - 1s: color=#4285F4

**效果**: Logo 带有弹性地放大并旋转，背景同时渐变

---

### 案例 2: 购物车添加动画

**需求**: 商品图片飞入购物车图标

**方案**:
1. **商品移动**: 使用 `Android Accelerate` (factor=2.0)
   - 0s: position=商品位置
   - 0.6s: position=购物车位置

2. **商品缩小**: 使用 `Android Accelerate` (factor=2.0)
   - 0s: scale=100%
   - 0.6s: scale=20%

3. **购物车弹跳**: 使用 `Rive Elastic` (amplitude=1.5, period=0.3)
   - 0.6s: scale=100%
   - 1s: scale=120%

**效果**: 商品加速飞入购物车，购物车弹跳反馈

---

### 案例 3: Material Design 卡片展开

**需求**: 卡片从列表展开为详情页

**方案**:
1. **卡片位置**: 使用 `Android FastOutSlowIn`
   - 0s: position=列表位置
   - 0.5s: position=屏幕中心

2. **卡片大小**: 使用 `Android FastOutSlowIn`
   - 0s: scale=50%
   - 0.5s: scale=100%

3. **内容淡入**: 使用 `Android LinearOutSlowIn`
   - 0.2s: opacity=0%
   - 0.6s: opacity=100%

**效果**: 符合 Material Design 规范的卡片展开动画

---

### 案例 4: iOS 通知横幅

**需求**: 通知从顶部滑入并停留

**方案**:
1. **滑入**: 使用 `iOS Spring Default`
   - 0s: y=-100
   - 0.5s: y=50

2. **停留**: 无动画
   - 0.5s - 3s: y=50

3. **滑出**: 使用 `iOS EaseIn`
   - 3s: y=50
   - 3.3s: y=-100

**效果**: 通知带有弹性地滑入，停留后快速滑出

## 效果对比

### 对比 1: 按钮点击反馈

| 曲线 | 效果描述 | 适用场景 |
|------|----------|----------|
| **Linear** | 匀速缩放，机械感 | 不推荐用于按钮 |
| **EaseOut** | 快速开始，缓慢停止 | 简单的按钮反馈 |
| **Overshoot** | 超过目标后回弹 | 强调点击效果 |
| **Elastic** | 多次振荡 | 有趣的交互反馈 |
| **Spring Bouncy** | 明显弹跳 | iOS 风格的反馈 |

**推荐**: `Overshoot` (tension=2.0) 或 `Elastic` (amplitude=1.2, period=0.3)

---

### 对比 2: 页面切换

| 曲线 | 效果描述 | 适用场景 |
|------|----------|----------|
| **Linear** | 匀速移动 | 简单场景 |

| **EaseInOut** | 缓入缓出 | iOS 标准 |

- Android 项目: `FastOutSlowIn`

## 参数调优指南

### Elastic 曲线调优

#### 振幅 (Amplitude)

| 值 | 效果 | 适用场景 |
|----|------|----------|
| 0.5 - 1.0 | 轻微振荡 | 细腻的 UI 动画 |
| 1.0 - 2.0 | 明显振荡 | 按钮反馈、强调动画 |
| 2.0 - 5.0 | 强烈振荡 | 游戏效果、夸张动画 |
| 5.0+ | 极端振荡 | 特殊效果 |

#### 周期 (Period)

| 值 | 效果 | 适用场景 |
|----|------|----------|
| 0.1 - 0.3 | 快速振荡 | 短时间动画 |
| 0.3 - 0.6 | 中速振荡 | 通用场景 |
| 0.6 - 1.0 | 慢速振荡 | 长时间动画 |
| 1.0+ | 极慢振荡 | 特殊效果 |

#### 推荐组合

```
轻快弹性: amplitude=1.0, period=0.2
标准弹性: amplitude=1.5, period=0.5
强烈弹性: amplitude=2.5, period=0.8
```

---

### Anticipate/Overshoot 调优

#### 张力 (Tension)

| 值 | 效果 | 适用场景 |
|----|------|----------|
| 0.5 - 1.0 | 轻微效果 | 细腻动画 |
| 1.0 - 2.5 | 明显效果 | 通用场景 |
| 2.5 - 4.0 | 强烈效果 | 夸张动画 |
| 4.0+ | 极端效果 | 特殊效果 |

**推荐值**: `2.0` (Android 默认值)

---

### Spring Custom 调优

#### 阻尼 (Damping)

| 值 | 效果 | 弹跳次数 |
|----|------|----------|
| 0.9 - 1.0 | 几乎无弹跳 | 0-1 次 |
| 0.7 - 0.9 | 轻微弹跳 | 1-2 次 |
| 0.5 - 0.7 | 明显弹跳 | 2-3 次 |
| 0.3 - 0.5 | 强烈弹跳 | 3-5 次 |
| 0.1 - 0.3 | 极端弹跳 | 5+ 次 |

#### 初速度 (Velocity)

| 值 | 效果 | 适用场景 |
|----|------|----------|
| 0.0 | 无初速度 | 从静止开始 |
| 0.5 - 1.0 | 轻微初速度 | 快速响应 |
| 1.0 - 2.0 | 明显初速度 | 强调动画 |
| 2.0+ | 强烈初速度 | 特殊效果 |

#### 推荐组合

```
细腻弹性: damping=0.85, velocity=0.0, duration=0.4
标准弹性: damping=0.7, velocity=0.0, duration=0.5
强烈弹性: damping=0.5, velocity=1.0, duration=0.6
快速响应: damping=0.75, velocity=0.8, duration=0.3
```

---

### Accelerate/Decelerate 调优

#### 因子 (Factor)

| 值 | 效果 | 适用场景 |
|----|------|----------|
| 0.5 - 1.0 | 轻微加速/减速 | 细腻动画 |
| 1.0 - 1.5 | 标准加速/减速 | 通用场景 |
| 1.5 - 2.5 | 强烈加速/减速 | 快速动画 |
| 2.5+ | 极端加速/减速 | 特殊效果 |

**推荐值**: `1.0` (Android 默认值)

## 总结

### 快速选择指南

**我想要...**

- **自然的移动**: `AccelerateDecelerate`, `EaseInOut`
- **快速响应**: `FastOutSlowIn`, `Decelerate`
- **有趣的反馈**: `Elastic`, `Overshoot`, `Spring Bouncy`
- **强调效果**: `Bounce`, `AnticipateOvershoot`, `Elastic`
- **简单动画**: `Linear`, `EaseIn`, `EaseOut`
- **Material Design**: `FastOutSlowIn`, `FastOutLinearIn`, `LinearOutSlowIn`
- **iOS 风格**: `Spring Default`, `EaseInOut`, `CAMediaTiming Default`

| 元素退出 | FastOutLinearIn | EaseIn |

| 强调动画 | Bounce | Spring Bouncy |
