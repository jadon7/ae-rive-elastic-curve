## 问题已修复 ✅

**v1.0.5 版本已修复关键帧区间计算问题！**

## 使用步骤

### 1. 运行脚本

在 After Effects 中：
```
文件 > 脚本 > 运行脚本文件 > 选择 RiveElasticCurve-standalone.jsx
```

✅ UI 窗口会立即显示在屏幕中央

### 2. 准备关键帧

1. 创建合成
2. 添加图层
3. 为属性（位置、缩放等）添加至少 2 个关键帧
4. 选中该属性

### 3. 应用弹性曲线

1. 在插件 UI 中调整参数：
   - 振幅 (Amplitude): 1.0 - 2.0
   - 周期 (Period): 0.3 - 0.7
   - 缓动类型: Ease Out（推荐）

2. 点击"应用到选中关键帧"

3. 播放预览查看效果

## 推荐参数

### UI 按钮弹跳
```
振幅: 1.5
周期: 0.4
缓动: Ease Out
```

### 物体落地反弹
```
振幅: 2.0
周期: 0.6
缓动: Ease Out
```

### 弹簧效果
```
振幅: 3.0
周期: 1.0
缓动: Ease In-Out
```

## 故障排除

### UI 不显示？
- 确保使用 v1.0.5 或更高版本
- 使用"运行脚本文件"方式运行
- 查看 `UI_FIX_DOCUMENTATION.md` 了解详情

### 应用失败？
- 确保选中了属性（点击属性名称）
- 确保属性有至少 2 个关键帧
- 确保打开了合成
